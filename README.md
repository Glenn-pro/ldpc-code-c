# LDPC (1023,781) C 語言實作

以 **C 語言**實作 **LDPC (1023,781)** 編碼與解碼模擬，並在 **AWGN 通道**下觀察 **BER / BLER** 的效能表現。

## 專案簡介

專案建立了一套完整的 LDPC (1023,781) 編解碼模擬流程，內容涵蓋矩陣資料讀取、編碼與解碼、通道雜訊模擬、錯誤率統計，以及結果輸出。

專案的重點不只是將通訊理論公式轉換為可執行的程式，更著重於實際工程開發中常見的問題處理，例如資料存取、記憶體管理、bit-level 邏輯運算、流程除錯，以及結果驗證。

## 主要功能

- 讀取 LDPC (1023,781) 矩陣資料
- 實作編碼與解碼流程
- 建立 AWGN 通道模擬
- 統計 BER / BLER 結果
- 輸出模擬結果與圖表

## 技術重點

- **C 語言演算法實作**
- **pointer 操作與陣列存取**
- **動態記憶體配置與釋放**
- **bitwise / XOR / parity 邏輯處理**
- **稀疏矩陣資料讀取與使用**
- **多層迭代流程除錯**
- **邊界條件檢查與結果驗證**

## 專案價值

本專案的價值不僅在於完成 LDPC 編解碼模擬，更在於實際處理了許多開發過程中會遇到的工程問題，包括：

- pointer 與陣列索引的正確存取
- 動態記憶體的配置、使用與釋放
- XOR / parity 等 bit-level 邏輯運算
- 稀疏矩陣資料的讀取與運用
- 多階段模擬流程的整合與除錯
- 模擬結果合理性檢查與流程修正

透過這個專案，我將通道編碼理論轉換為可執行的 C 程式流程，並練習如何從數學模型、資料結構、程式控制流程到結果驗證，完整建立一個可運行的模擬系統。
